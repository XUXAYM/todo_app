# todo_app

Application for storing your todos.

## Features list

* Adding "todo" notes;
* Deleting "todo" notes;
* Editing "todo" notes;
* Online data synchronization between devices;
* Works offline;
* Dark/light theme support;
* Interface internationalization (Russian, English).
